Welcome to My Projects Page

Here are some of the projects I have worked on using the Laravel framework:


1. Face Recognition Course Attendance System

This is a web-based attendance system that utilizes face recognition technology to automate student attendance tracking in university courses.

  • Framework: Laravel
  • Technology: OpenCV, PHP, MySQL
  • Features: Face scanning, real-time attendance records, reports.

2. H2OSaver - Water Usage Tracker

H2OSaver is a web application designed to help users track and minimize their water usage. The system provides insights, reports, and recommendations for reducing water waste.

  • Framework: Laravel
  • Technology: PHP, JavaScript, MySQL
  • Features: Water usage tracking, goal setting, data visualization.
